JavaScript (JS)

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

are used to assign values, compare values, perform arithmetic operations, and more.

Variables

are containers for storing data values.

Variable naming conventions

a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

Functions

a block of code designed to perform a particular task.

Methods

actions you can perform (on HTML Elements).

Arguments

the real values passed to (and received by) the function.

Parameters

the names listed in the function definition.

Return

stops the execution of a function and returns a value from that function.

Chaining methods

allows us to run multiple jQuery methods (on the same element) within a single statement.

Strings

stores a series of characters

Booleans

represents either true or false

Undefined

indicates that a variable has not been assigned a value.

NaN

used when a variable or value does not equate to a number.

Escape

computes a new string in which certain characters have been replaced by a hexadecimal escape sequence.

Alert

displays an alert box with a specified message and an OK button.

Comments

can be used to explain code, and to make it more readable.

jQuery

a cross platform javaScript library.

Attributes

provide additional information about HTML elements.
